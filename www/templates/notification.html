In this section we are going to see how to create notification.

First of all we need to install cordova plugin : cordova plugin add https://github.com/katzer/cordova-plugin-local-notifications.git After that if you already have configured ngCordova by adding ng-cordova.js and including ngCordova to your app, you can use $cordovaLocalNotification in your controller

You need to create 3 buttons :

  • - A button that raises a notification with an image.
  • - A button that schedule a notification.
