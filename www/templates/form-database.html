In this section you need to create a form. On form submit you'll save data in database. After that, change dataService to use database informations.

Steps :
  • 1 - Add ngCordova (bower)
  • 2 - Add add $cordovaSQLite plugin (cordova)
  • 3 - Add init database in app.js (use $ionicPlatform.ready)
  • 4 - Create a database service to execute query
  • 5 - Add insert and get methods to dataService using databaseService.
  • 6 - Create a form to insert new items.
  • 7 - Change simple list data to load database items
